# Flight Data Analysis — Big Data (CS644)

**Course:** CS644 Big Data — NJIT  
**Team:** Antonita Racheal, Jaime, Soumya, Surbhi  
**Tools:** Java, Hadoop, MapReduce, Apache Spark

## Overview
Group project analyzing large-scale U.S. flight datasets to uncover patterns 
in delays, cancellations, and route performance using distributed big data 
processing techniques.

## What This Does
- Processes large flight datasets using **Hadoop MapReduce** in Java
- Analyzes flight delay patterns, cancellation rates, and carrier performance
- Compiled into a runnable `.jar` for cluster deployment
- Full technical report included as PDF

## Files
| File | Description |
|------|-------------|
| `flightproject.java` | Main Java MapReduce program |
| `flightproject.jar` | Compiled executable for Hadoop cluster |
| `command.txt` | Commands used to run the job |
| `output.txt` | Sample output results |
| `job.properties` | Job configuration |
| `Big Data Final Project.pdf` | Full technical report |

## Tech Stack
Java · Hadoop · MapReduce · Apache Spark · Distributed Computing
